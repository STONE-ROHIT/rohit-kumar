# Personal Portfolio Website

A responsive personal portfolio website built using **HTML**, **CSS**, and **Vanilla JavaScript**.  
This website is used to showcase my projects, skills, and professional profile.

The goal of this project was to practice core web fundamentals and create a clean,
simple platform to present my backend and programming work.

---

## Features

- Responsive design for desktop and mobile devices
- Smooth navigation between sections
- Projects section to showcase work
- Contact section with quick access to email and social links
- Subtle animations for improved user experience
- No frontend frameworks used

---

## Tech Stack

This project is built entirely using core web technologies:

- **HTML5** – Structure and semantic markup
- **CSS3** – Styling and responsive layout
- **JavaScript (Vanilla)** – Interactivity and DOM manipulation
- **Swiper.js** – Carousel / slider functionality
- **GSAP** – Basic animations
- **Remix Icons** – Icons used across the website

> No frameworks like React, Angular, or Vue are used.  
> The website is built from scratch using plain HTML, CSS, and JavaScript.
